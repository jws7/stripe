# Instructions
1) Clone repo
2) cd stripe/
3) mvn package
4) java -cp target/spark-server-0.0.1-SNAPSHOT-jar-with-dependencies.jar net.jws7.Server
5) load web/checkout.html in a browser
6) Test credit card payment
7) orders.txt appears in stripe/
